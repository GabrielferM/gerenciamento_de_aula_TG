# gerenciamento_de_aula_TG

---
Grupo:

Gabriel Ferreira Marra

O projeto Gerenciamento de Aula tem como objetivo desenvolver um sistema web voltado à organização e planejamento de aulas, permitindo ao usuário estruturar conteúdos futuros, acompanhar o andamento das aulas e visualizar projetos e atividades de forma clara e organizada por meio de roadmaps e calendário.

O sistema busca centralizar informações acadêmicas, facilitando o controle do que já foi ministrado, do que ainda será abordado e de como os projetos estão distribuídos ao longo do tempo e separação de equipes.

Tecnologia:

-Front-end
    React

-Back-end
    Nest.js

-Banco de dados
    Prisma
    SqLite
